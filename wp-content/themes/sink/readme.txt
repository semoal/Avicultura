== Description ==
Sink - Minimal WooCommerce WordPress Theme.
It is readable on a wide variety of screen sizes, and suitable for multiple languages.
We designed it using a mobile-first approach, meaning your content takes center-stage, regardless of whether your visitors arrive by smartphone, tablet, laptop, or desktop computer.

* Responsive Layout

* Menu Description
* Post Formats

For more information about Sink please go to http://themeforest.net/user/themehippo

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button and click 'Upload Theme'.
2. Extract .zip file and upload 'sink.zip' file
3. Click on the 'Activate' button to use your new theme right away.
4. Go to http://www.cloudsoftwaresolution.com/documentation/wordpress/sink/ for a guide on how to customize this theme.
5. Navigate to 'Sink Options' in your admin panel and customize to taste.
